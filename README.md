# 🏗️ Fundamentals of Web Development
> **"Give me six hours to chop down a tree and I will spend the first four sharpening the axe."**

## 🧠 The Philosophy
In an era where the industry is racing to ship products at breakneck speed, I am intentionally choosing a different path. While others are moving fast, I am slowing down to master the **fundamentals**.

### Why am I doing this?
* **AI with Intent:** I don't want to just use AI to write code I don't understand. I want to build my fundamentals so that when I use AI, I know exactly what is happening under the hood, why it works, and how to fix it when it breaks.
* **Deep Roots:** I am currently "half-knowledgeable," but I am bridging that gap by focusing on the core mechanics of the web rather than just chasing the latest frameworks.
* **Quality over Speed:** Understanding the *why* behind the code is my priority. This repository is my digital garden where I document that journey.

---

## 📂 Repository Roadmap
This repo is a collection of my deep dives into the building blocks of the web.

### 🛠️ Current Focus Areas
* **Core JavaScript:** Moving beyond syntax to understand Execution Context, Closures, and Asynchronous patterns.
* **DOM Mechanics:** Learning how the browser actually renders and manipulates elements without heavy abstractions.
* **CSS Architecture:** Mastering layouts (Flex/Grid) and the cascade before reaching for Tailwind or Bootstrap.
* **The Request/Response Cycle:** Understanding how data actually travels across the wire.
